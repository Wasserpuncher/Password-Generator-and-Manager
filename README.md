# PasswordGeneratorManager

PasswordGeneratorManager is a Python tool that generates strong passwords and securely manages them. It allows users to generate passwords, store them for different accounts, retrieve passwords, and encrypt them for security.

## Features

- **Password Generation**: Generates strong passwords with customizable length.
- **Password Storage**: Stores passwords securely using SHA-256 hashing.
- **Password Retrieval**: Retrieves stored passwords for different accounts.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/wasserpuncher/PasswordGeneratorManager.git
    cd PasswordGeneratorManager
    ```

## Usage

1. Run the password manager script:
    ```bash
    python password_manager.py
    ```
2. Follow the on-screen instructions to generate, store, and retrieve passwords.

## Security Note

This tool uses SHA-256 hashing for password storage, which is secure but not immune to all forms of attacks. Exercise caution and use strong passwords.
